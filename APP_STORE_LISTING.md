## 📝 앱 설명 (Full Description)

### Google Play Store (한국어)

```
🤖 BlogTwin - AI가 당신의 스타일로 블로그를 작성합니다

BlogTwin은 AI 기술을 활용하여 당신만의 글쓰기 스타일을 학습하고,
네이버 블로그 포스팅을 자동으로 작성해주는 혁신적인 앱입니다.

✨ 주요 기능

📚 스타일 학습
• 기존 블로그 글을 분석하여 당신의 글쓰기 스타일 학습
• 어조, 문체, 선호 주제 파악
• 카테고리별 특징 자동 추출

🎨 AI 자동 작성
• GPT-4 기반 고품질 콘텐츠 생성
• 카테고리별 맞춤 글 작성
• 실시간 생성 과정 확인

📸 사진 기반 포스팅
• 사진만 선택하면 스토리 자동 생성
• GPT-4 Vision으로 이미지 분석
• 이미지 설명 및 배치 자동 최적화

✍️ 리치 텍스트 에디터
• 직관적인 포맷팅 도구
• 링크, 이미지 삽입
• 자동 저장 기능

🤝 AI 어시스턴트
• 문장 다듬기
• 맞춤법 검사
• 표현 개선
• 톤 조정 (격식/캐주얼/전문적/친근함)

📤 간편한 발행
• 네이버 블로그 연동
• 공개 설정, 카테고리, 태그 관리
• 예약 발행 지원
• 클립보드 복사로 수동 발행

🎯 이런 분들께 추천합니다

• 블로그를 시작하고 싶지만 글쓰기가 어려운 분
• 시간이 부족해 블로그 업데이트가 어려운 분
• 일관된 스타일로 블로그를 운영하고 싶은 분
• 사진 위주의 포스팅을 자주 하시는 분
• AI 기술에 관심이 있는 블로거

💡 BlogTwin의 특별함

1️⃣ 진짜 당신의 스타일
   단순히 AI가 쓴 글이 아닌, 당신의 스타일을 학습한 AI가 작성한 글

2️⃣ 시간 절약
   30분 걸리던 글쓰기를 3분으로 단축

3️⃣ 품질 보장
   GPT-4 기반 고품질 콘텐츠 생성

4️⃣ 편리한 편집
   AI가 생성한 글을 쉽게 수정하고 개선

5️⃣ 네이버 블로그 최적화
   네이버 블로그에 최적화된 기능 제공

🔒 개인정보 보호

• 모든 데이터는 안전하게 암호화되어 저장됩니다
• 네이버 OAuth 2.0 공식 인증 방식 사용
• 블로그 콘텐츠는 로컬 및 암호화된 클라우드에만 저장
• 제3자와 정보를 공유하지 않습니다

📱 필요 권한

• 인터넷: 네이버 블로그 연동, AI 콘텐츠 생성
• 카메라: 사진 촬영 (선택)
• 저장공간: 이미지 선택 및 저장 (선택)

💬 문의 및 지원

버그 리포트나 기능 제안은 아래로 연락 주세요:
이메일: support@blogtwin.app
웹사이트: https://blogtwin.app

BlogTwin과 함께 블로그 운영을 더 쉽고 즐겁게!
```

### App Store (한국어)

```
BlogTwin - AI가 당신의 스타일로 블로그를 작성합니다

AI 기술로 당신의 글쓰기 스타일을 학습하고, 네이버 블로그 포스팅을 자동으로 작성해주는 혁신적인 앱입니다.

주요 기능:

📚 스타일 학습
기존 블로그 글을 분석하여 당신만의 글쓰기 스타일을 학습합니다. 어조, 문체, 선호 주제까지 파악하여 진짜 당신이 쓴 것 같은 글을 만들어냅니다.

🎨 AI 자동 작성
GPT-4 기반으로 고품질 블로그 포스트를 자동 생성합니다. 카테고리별 맞춤 글 작성, 제목 자동 생성, 해시태그 추천까지 모두 AI가 처리합니다.

📸 사진 기반 포스팅
사진만 선택하면 AI가 자동으로 스토리를 만들어줍니다. GPT-4 Vision이 이미지를 분석하고, 이미지 설명과 배치까지 최적화합니다.

✍️ 리치 텍스트 에디터
직관적인 에디터로 AI가 생성한 글을 쉽게 수정할 수 있습니다. 굵게, 기울임, 링크, 이미지 삽입 등 모든 기능을 지원합니다.

🤝 AI 어시스턴트
문장 다듬기, 맞춤법 검사, 표현 개선, 톤 조정 등 글쓰기를 도와주는 AI 어시스턴트가 함께합니다.

📤 간편한 발행
네이버 블로그와 연동하여 클립보드 복사만으로 쉽게 발행할 수 있습니다. 공개 설정, 카테고리, 태그 관리는 물론 예약 발행도 지원합니다.

BlogTwin과 함께 블로그 운영을 더 쉽고 즐겁게!
```
